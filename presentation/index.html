Django?

  • Django — свободный фреймворк для веб-приложений на языке Python, использующий шаблон проектирования MVC.
  • 2003 год — первая внутренняя версия фреймворка, в рамках проектов для газеты «Lawrence Journal-World»
  • 2005 год — выход в OpenSource, первый релиз на github

Парадигмы Python :

  • функциональная — процесс вычисления трактуется как вычисление значений функций в математическом понимании (способ решения задачи описывается в виде зависимости функций друг от друга)
  • императивная — описывает процесс вычисления в виде инструкций, изменяющих состояние программы
  • объектно-ориентированная — основной концепцией является понятие объекта, отождествляемого с объектом предметной области
